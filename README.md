# cljwc

wc command by clojure.

## Usage

### Run cljwc command
./bin/cljwc

### Options

```
-h, --help Show Help
-i, --input Input
-c, --bytes print the byte counts
-m, --chars Count characters
-l, --line  Count lines
```
